# vue-directive-dragon

`Vue` 拖拽指令，使用 `Vue` 版本为：`2.5.21`

当前状态：
1. 功能基本完成
2. 文档还未完成
3. 性能持续优化中

已实现功能：
1. 简单拖拽（支持设置可否超出边界）
2. 同数据源下上拖拽排序
3. 不同数据间相互拖拽和排序

待处理：
1. 多组件情况下，使用指令
2. 性能优化
